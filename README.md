# Vansh Rana - Personal Portfolio

Welcome to my personal portfolio repository! This is a static website containing information about my skills, experience, and past work.

## Structure
- `index.html`: Home page
- `experience.html`: My professional experience
- `skills.html`: Technologies and skills I work with
- `work.html`: My portfolio projects
- `contact.html`: How to reach me
- `Vansh_Rana_Resume.pdf`: My current resume
- Python scripts: Various utilities used to fix or format HTML elements.

## How to View
Simply open `index.html` in any modern web browser to view the portfolio locally.
